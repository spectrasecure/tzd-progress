# The Zeta Directive Progress

Public aggregate progress data for *The Zeta Directive*.

This repository is a projection target for the private manuscript repository. It intentionally contains only aggregate counts for public display on neosynth.net: no manuscript text, filenames, commit SHAs, branches, or internal structure.

## Data

- `progress.json` — current public progress snapshot.

```text
https://cdn.jsdelivr.net/gh/spectrasecure/tzd-progress@main/progress.json
```

Raw fallback:
